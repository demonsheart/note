# Dos命令



## 打开CMD方式

1. 开始+系统+命令提示符
2. Win+R 输入cmd
3. 任意文件夹下 shift+右键
4. 资源管理器的地址栏 前面+cmd



## 常用Dos命令

```bash
F: #盘符切换
dir #查看目录文件
cd /d E:\.... #切换目录 
cd .. #返回上一级
cls #清除屏幕
exit #退出
ipconfig #查看网络

# ping命令
ping www.baidu.com

#文件操作
md test #创建目录
rd test #删除目录
cd>a.txt #新建文件
del a.txt #删除文件
```
