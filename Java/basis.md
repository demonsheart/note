# Java基础



## 编译运行

1. 编写Hello.java

   ```java
   public class Hello {
       public static void main(String[] args) {
           System.out.print("Hello,World!");
       }
   }
   ```

   注意类名和文件名保持一致并且首字母大写

2. 编译.java文件

   ```bash
   javac Hello.java
   ```

   得到一个Hello.class文件

3. 运行

   ```bash
   java Hello
   ```

   [如果出现错误，参考](https://www.zhihu.com/question/36537093)
